# Database Files

This folder contains database artifacts used by the project.

Files included:

- `221775_221847_222679_DBProject_Attendance_Portal.sql` - primary database script.
- `Attendance_Database_Model.accdb` - Microsoft Access database model.
- `xampp sql data.rar` - XAMPP import data archive.
- `xampp.rar` - XAMPP installer/archive.

Use these files to restore the MySQL database or review the database schema.
