# React App

The frontend for the Attendance and Leave Management Portal.

## Overview

This folder contains the React + Vite web application used for the portal interface.

## Run locally

```bash
cd react-app
npm install
npm run dev
```

The development server will usually start at `http://localhost:5173`.

## Project structure

- `src/` - source files for the React application.
- `src/Components/` - individual UI components for login, registration, and portal screens.
- `tailwind.config.js` - Tailwind CSS configuration.
- `vite.config.js` - Vite build setup.

## Notes

The React app is designed to work with the backend API in `react-server/`. Update any API base URLs in the source if your server is hosted on a different address.
